# siaf_sync
## Acerca de Siaf Sync
Herramienta para extraer datos de las tablas de SIAF y disponer de informacion para el armado de reportes personalizados a demanda de las areas administrativas de los gobiernos Locales, Regionales y entidades publicas.

# Funcionalidades 
- Leer a modo de lectura los archivos *dbf
- La lectura de *.dbf no bloquea ninguno de ellos, ya qiu solo las leee en segundos
